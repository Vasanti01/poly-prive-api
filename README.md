# Poly Privé AI API

Backend API powering Poly Privé AI.

## Features

- Wallet Portfolio
- Native Balance
- ERC20 Tokens
- Portfolio Value
- Multi-chain Support
- Moralis Integration

## Install

```bash
npm install
```

## Start

```bash
npm run dev
```

## API

### Health

```
GET /health
```

### Wallet

```
GET /wallet/:address
```

Example

```
GET /wallet/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045
```
